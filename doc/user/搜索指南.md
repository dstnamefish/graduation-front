1. 搜“人”（最常用）
如果你要找某个患者、某个医生或某个同事，这是最简单的办法。

患者：直接输入患者的姓名或病历号。

例如：张三、0000123456

医护人员：输入医生的姓名，可以查到他的排班、简介或联系方式。

例如：李医生、王主任

内部员工：输入同事的名字，可以查到科室电话或邮箱。

2. 搜“病”与“药”（临床最需要）
如果你是在临床工作中需要快速查阅资料：

疾病/症状：输入症状或疾病名称，系统会关联出相关的诊疗指南、以往类似病例或健康宣教资料。

例如：糖尿病、发热、高血压

药品：输入药品的通用名或商品名，可以查到说明书、库存情况、价格以及有无可替代的院内药品。

例如：阿司匹林、胰岛素

3. 搜“地点”（导诊/后勤最需要）
如果你是在找路，或者需要去某个地方执行任务：

科室/检查室：输入科室名称，可以弹出地图导航或楼层指引。

例如：抽血处在哪？、CT室、住院部

设施/设备：输入具体设备名称，可以查到它目前在哪个病房，或者它的维修保养记录（后勤专用）。

例如：呼吸机、床旁B超机

4. 搜“事”与“流程”（管理最需要）
如果你是在处理行政事务或学习规范：

规章制度：输入某个流程的关键词，可以找到相关的红头文件、SOP操作流程或应急预案。

例如：请假流程、报销标准、手卫生规范

工作安排：输入“排班表”、“会议通知”等，可以找到近期的安排。

5. 搜“钱”与“物”（收费/库房最需要）
收费项目：输入治疗或检查名称，可以查到价格和医保报销比例。

例如：核磁共振多少钱？、换药费

耗材/物资：输入耗材名称，可以查到库存位置和数量。

例如：一次性口罩、采血管
