## 项目结构
前端项目采用了模块化的目录结构，包括：

- src/app ：应用核心配置，包括布局、状态管理等
- src/entities ：领域实体模型
- src/features ：功能模块，按业务功能划分
- src/locales ：国际化资源
- src/mock ：模拟数据
- src/pages ：页面组件
- src/plugins ：插件配置
- src/router ：路由配置
这种结构遵循了现代前端项目的最佳实践，有利于代码的组织和维护。

## 技术特点
1. 类型安全 ：使用 TypeScript 提供全面的类型检查
2. 组件化 ：基于 Vue 3 的组件系统，实现代码复用
3. 响应式 ：利用 Vue 3 的响应式系统，实现数据与视图的自动同步
4. 模块化 ：清晰的目录结构和模块划分
5. 现代化构建 ：使用 Vite 提供快速的开发体验和优化的生产构建
6. 工具链完善 ：集成了 ESLint、Prettier 等工具，保证代码质量
7. 性能优化 ：使用 @tanstack/vue-query 进行数据缓存，减少重复请求
8. 用户体验 ：集成了动画、进度条等增强用户体验的库
整体技术栈选择现代、成熟且性能优异，适合构建复杂的医疗系统前端应用。
